## 📝 使用說明

### 統一指令格式

所有指令都支持以下格式：

3. **帶參數**：`/交易所 買入 魚乾 10`


## 🔧 技術說明

### 指令分發機制

複雜的指令（如交易所、骰寶、紅包）使用分發器模式：
- 主指令作為入口
- 子命令通過參數分發到對應 handler

### 別名處理

- 所有別名都支持繁簡中文自動轉換
- 別名在 `main.py` 中通過 `@filter.command(alias=[...])` 定義
- 指令處理時會自動標準化為對應的 handler
